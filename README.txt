BUILDING AND RUNNING CODE (Python 3)
----------------------------------------------------------------

kmeans_twitter.py :

1. Extract files from the zip file
2. Launch Google Colab (or Jupyter Notebook)
3. Open kmeans_twitter.py in a google notebook (or a jupyter notebook)
4. (Optional) In the main function, change the value passed to the KMeans object to set the desired k value and number of iterations
5. Run the program

OR

1. Run via command line: "python3 kmeans_twitter.py"


LIBRARIES USED
----------------------------------------------------------------

- numpy
- pandas
